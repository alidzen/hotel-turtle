Кнопки

Описание

У кнопки обязательно должен быть border. Если в дизайне он не предусматривается, то делайте его прозрачным transparent. Так вы предостерегаете себя от того, чтобы кнопка «прыгала», когда появится эффект наведения на border у кнопки.

Использование

Jade

  .b-btn

Пример jade

  a(href="#").b-btn Ссылка
  input(type="submit" value="Отправить").b-btn

Пример html

  <a href="#" class="b-btn">Ссылка<a>
  <input type="submit" class="b-btn" value="Отправить">

Параметры

Имя Описание Тип Зн. По-умолчанию
$btn-color Цвет текста color - $white
$btn-bg-color Цвет фона color - $black
$btn-bd-width Размер бордера number px 1px
$btn-bd-color Цвет бордера color - transparent
$btn-border-radius Радиус скругления number px 3px
$btn-font-size Размер текста number px $base-font-size
$btn-line-height Межстрочный интервал number px $base-line-height
$btn-font-family Шрифт string - $font
$btn-height Высота кнопки number rhythm 2
$btn-tt-uppercase Преобразование текста в прописные или заглавные bool false/true false

Дополнительные параметры — hover effect

Имя Описание Тип Зн. По-умолчанию
$btn-color-hover Цвет текста color - $black
$btn-bg-color-hover Цвет фона color - $white
$btn-bd-color-hover Цвет бордера color - transparent

Дополнительные параметры — disabled

Для полей ввода input[type=submit], button нужно добавить аттрибут disabled
Для всего остального класс .is-disabled

Имя Описание Тип Зн. По-умолчанию
$btn-color-disabled Цвет текста color - #ccc
$btn-bg-color-disabled Цвет фона color - #eee
$btn-bd-color-disabled Цвет бордера color - transparent

Модификаторы

Имя Описание
.b-btn_width_auto Устанавливает размер кнопки равным размеру контента.
.b-btn_tt_uppercase Преобразование текста в прописные или заглавные.

Высота кнопки

Так как мы используем вертикальный ритм в проекте, то мы заранее знаем высоты кнопок.

Настройкой высот является массив $btn-list-height из пар значений название/высота в ритмах.

Название Высота в ритмах
small .5
medium 1
big 1.5
large 2

На основе этого массива создаются классы

.b_btn_height_small
.b_btn_height_medium
.b_btn_height_big
.b_btn_height_large

Названия и высоты в массиве вы можете изменять/добавлять.
